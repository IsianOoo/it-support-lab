# it-support-lab

IT Support Lab

To repozytorium zawiera dokumentację ćwiczeń wykonanych w ramach nauki pracy na stanowisku IT Support / Helpdesk.
Zadania zostały wykonane w środowisku testowym i odzwierciedlają typowe zgłoszenia spotykane w firmach świadczących outsourcing IT.

Zakres prac obejmuje wsparcie użytkowników końcowych, diagnostykę systemów Windows, podstawy sieci komputerowych oraz obsługę usług Microsoft 365 na poziomie helpdesk.

Środowisko testowe:
- Windows 11 (maszyna wirtualna – VirtualBox)
- Sieć: NAT, DHCP, DNS
- Microsoft 365 (tenant testowy)
- Konto użytkownika bez uprawnień administracyjnych + konto administracyjne do celów diagnostycznych

W repozytorium znajdują się opisane przypadki zgłoszeń (tickety), zawierające:
- opis problemu zgłoszonego przez użytkownika,
- kroki diagnostyczne,
- identyfikację przyczyny,
- zastosowane rozwiązanie.

Ćwiczenia skupiają się na logicznej analizie problemu i poprawnym procesie diagnostycznym, a nie na automatycznym stosowaniu pojedynczych komend.

Struktura:
- tickets/ – dokumentacja poszczególnych przypadków (Markdown)

Zakres zagadnień:
- konfiguracja i diagnostyka połączeń sieciowych (IP, DHCP, DNS, gateway),
- podstawowe narzędzia diagnostyczne (ipconfig, ping, nslookup),
- rozwiązywanie problemów użytkowników systemu Windows,
- dokumentowanie rozwiązań w formie ticketów.

Celem repozytorium jest zebranie praktycznych przykładów pracy IT Support, które mogą służyć jako materiał referencyjny oraz uzupełnienie doświadczenia praktycznego.
